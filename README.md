````markdown
## üöÄ Getting Started

### Prerequisites

To run this project locally, you‚Äôll need:

- **Flutter SDK**: [Download Flutter](https://flutter.dev/docs/get-started/install)

   ```bash
````

   ```bash

   ```
3. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

4. **Run the App**:
   ```bash
   flutter run
   ```
